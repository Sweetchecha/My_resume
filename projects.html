Мои проекты

1. Чат-бот для Telegram

Чат-бот, разработанный с использованием Python и библиотеки aiogram. Поддерживает автоматическое реагирование на команды, предоставляет данные из различных API, включая погодные данные и курсы валют. Интегрирован с базой данных SQLite для хранения пользовательских настроек и истории запросов.

2. Веб-приложение для управления задачами

Веб-приложение, созданное с использованием Django. Позволяет пользователям регистрироваться, создавать задачи, отслеживать их статус и делиться задачами с другими пользователями. Использована PostgreSQL для хранения данных и Bootstrap для создания адаптивного интерфейса.

3. API для обработки изображений

RESTful API, разработанный на FastAPI, которое предоставляет функционал для обработки изображений, включая изменение размера, применение фильтров и конвертацию форматов. Хранение изображений осуществляется с использованием облачного хранилища, а документация API автоматически генерируется с помощью Swagger.

4. Проект машинного обучения: Анализ настроений

Создано приложение, которое анализирует настроения текстов с помощью алгоритмов машинного обучения. Использованы библиотеки Scikit-learn и TensorFlow для построения моделей классификации. В проекте собран большой датасет из Twitter, и результаты визуализированы с помощью Matplotlib и Seaborn. Также разработан веб-интерфейс для загрузки текстов и отображения анализа в реальном времени.

5. Система мониторинга серверов

Программа на Python для мониторинга состояния серверов. Позволяет отслеживать использование процессора, оперативной памяти, дискового пространства и сетевого трафика. Уведомления о сбоях отправляются через Telegram-бот. Использованы библиотеки psutil, asyncio и smtplib для интеграции с email.
